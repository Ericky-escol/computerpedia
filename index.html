O que é um Software

Software é um conjunto de instruções, programas e dados que permitem que um computador, smartphone ou qualquer dispositivo eletrônico realize tarefas específicas. Diferente do hardware, que é a parte física do dispositivo, o software é intangível, mas podemos usar e interagir com ele.

Um sistema é um conjunto de componentes físicos ou lógicos interligados para atingir objetivos específicos.

O que é um Malware

Malware vem de "malicious software" (“software malicioso”) é qualquer programa criado com a intenção de prejudicar, explorar ou acessar sistemas sem autorização. Não são limitados apenas a computadores: smartphones, tablets, servidores, até dispositivos de IoT (Internet das Coisas) podem ser infectados.

O malware geralmente precisa enganar o usuário para entrar no sistema, ou explorar falhas no software. Alguns métodos comuns de infecção são:

  • Download de programas ou arquivos
  • Abertura de anexos de e-mails ou mensagens suspeitas
  • Exploração de falhas de sistemas
  • Downloads automáticos através de sites maliciosos
